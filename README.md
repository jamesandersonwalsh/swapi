# SWAPI

A small project for playing around with SWAPI _(Star Wars API)_

Written with React + Typescript + Vite

## Getting Started

```bash
pnpm i && pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
